# student-management-system-c
A command-line Student Management System in C using file handling to add, view, search and delete student records. 

## Features:
-> Add Student.
-> View all sstudents.
-> Searh student by id.
-> Delete student.

## Technologies Used:
-> C Programming
-> File Handling

## How to run 
1. Compile: g++ file name.cpp -o ev
2. Run: ev.exe
